# Titanic Survival Analysis

This project analyzes the Titanic dataset to explore survival patterns and build a machine learning model to predict survival.  

- 📊 Dataset: 891 passengers from the Kaggle Titanic dataset  
- 🔎 Methods: Data cleaning, EDA, feature engineering, Logistic Regression  
- 📈 Result: Accuracy ~76%; gender and class were strong predictors of survival & Age and PassengerId showed almost no influence.
- 🛠️ Tools: Python, pandas, matplotlib, seaborn, scikit-learn
